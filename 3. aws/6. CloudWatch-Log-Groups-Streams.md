### Log Groups vs Log Streams
```
Log Group:
- Container for related log streams
- Defines retention period
- Sets access permissions  
- Configures metric filters
- Groups logs by application/service

Examples:
/aws/codebuild/my-web-app-build
/aws/lambda/user-authentication
/aws/apigateway/my-api
/myapp/frontend
/myapp/backend

Log Stream:
- Sequence of log events
- From single source (instance, container, build)
- Ordered chronologically
- Cannot span multiple sources

Examples:
2024/02/03/[$LATEST]abc123def456  (Lambda)
i-0123456789abcdef0               (EC2)
build-a1b2c3d4-e5f6-7890          (CodeBuild)
web-server-pod-xyz789             (EKS)
```
